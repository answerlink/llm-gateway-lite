## 🔒 安全检查

在推送到 GitHub 前，确保：

- [ ] **没有提交敏感信息**
  - [ ] `configs/providers.yaml` 不在 Git 中（在 .gitignore 里）
  - [ ] `configs/models.yaml` 不在 Git 中（在 .gitignore 里）
  - [ ] 日志文件不在 Git 中（`logs/*.log`）
  
- [ ] **示例文件已清理**
  - [ ] `.example` 文件中没有真实的 API Keys
  - [ ] `.example` 文件使用占位符（如 `sk-your-api-key-here`）

### 快速检查命令

```bash
# 检查是否有敏感信息
grep -r "sk-[a-zA-Z0-9]" configs/ --include="*.yaml" --exclude="*.example"

# 确保示例文件存在
ls -la configs/*.example

# 确保配置文件被忽略
git status --ignored | grep configs/providers.yaml
git status --ignored | grep configs/models.yaml
```
